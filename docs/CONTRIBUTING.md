## Prerequisites

- node (>= 6.2)

## Install

```bash
git clone https://github.com/tonaljs/tonal
npm run install
npm run build
```

## Develop

Ensure all tests pass:

```
npm run test
```

Use the libraries from node repl:

```bash
> S = require('./packages/scale');
> S.scale('C chromatic');
```
