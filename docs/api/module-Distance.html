Distance

Transpose notes by intervals and find distances between notes

Source:
  • packages/distance/index.js, line 1
Examples
// es6
import * as Distance from "tonal-distance"
Distance.interval("C3", "C4") // => "1P"
// es6 import selected functions
import { interval, semitones, transpose } from "tonal-distance"

semitones("C" ,"D") // => 2
interval("C4", "G4") // => "5P"
transpose("C4", "P5") // => "G4"
// included in tonal facade
const Tonal = require("tonal");
Tonal.Distance.transpose("C4", "P5")
Tonal.Distance.transposeBy("P5", "C4")

Find the interval between two pitches. It works with pitch classes (both must be pitch classes and the interval is always ascending)

(static) transpose(note, interval) → {String}

Transpose a note by an interval.

Source:
  • packages/distance/index.js, line 95

Transpose a note by an interval. The note can be a pitch class.

This function can be partially applied.

Example
import { tranpose } from "tonal-distance"
transpose("d3", "3M") // => "F#3"
// it works with pitch classes
transpose("D", "3M") // => "F#"
// can be partially applied
["C", "D", "E", "F", "G"].map(transpose("M3)) // => ["E", "F#", "G#", "A", "B"]
